> **Driver L2 thật** tiêu thụ fabric này: **OpenVPN tap-mode (V2.h)** và **SoftEther (V4.c)** — cả hai có **hai chế độ**: (1) bắc-cầu-1-host mặc định (`OpenVpnTapChannel`/`SoftEtherEthernetChannel`→`VirtualHost`+`ArpResolver`→`IPacketChannel`), và (2) **multi-host data-plane** opt-in: data channel gắn vào `EthernetSwitch`/`EthernetAdapter` như **uplink port** (`ConnectUplink`), `MultiHostSession` (L2.8) phơi N station = N `IVpnSession` (mỗi station DHCP lease riêng do server cấp), thêm station qua `IVpnConnection.OpenSessionAsync`.
